JavaScript Object.values()

Back to Dashboard

The Object.values() method returns the values of an object.
If you wanted to find out the values of the object so this is the best method to find out the values.

Note: The Object.values() method does not change the original object.

Related Methods:

Object.keys() returns the keys (properties) of any object type.

Object.values() returns the values of all object keys (properties).

Object.entries() returns the keys and values of any object types.

The methods above return an Iterable (enumerable array).

Iterables makes it simpler to use objects in loops and to convert objects into maps.

While Passing the Array inside Object.values() method

Syntax:
Object.values(obj);
Parameter:
  • obj: It is the object whose enumerable properties are to be returned.
Remember few points:
  • O is the capital letter while typing Object.values()
  • v is the small letter while typing .values()
  • It will always return the array
  • If you are passing the object inside the Object.values() then it will return the values within the array
  • And if you are passing the array inside the Object.values then it will return the values within the array
Return Value:

It Returns an array containing all the enumerable property values of the given object.
