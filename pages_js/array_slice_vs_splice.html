What is the difference slice & Splice ?

Slice:

  • Slice method is always returning a new Array because Array.slice() does not change the original array.
  • You need to provide the starting and ending index of array then it will select that come up with the new array.

The slice() method returns the selected elements in an array, as a new array object.

This is a Syntax: array.slice(start, end)

Splice:

  • Splice method is also using for slicing of your array values But Splice method does not returning a new Array its producing new value in an Existing array.
  • It does not generate a new array
  • If you want to add or delete the array values within Existing array without using pop(), push(). shift(), unshift() so you can perform the Splice method for solving your problems.

Syntax: splice(index, how many delete, add new value);
