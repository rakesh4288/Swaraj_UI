Hoisting Concept in JavaScript

Back to Dashboard

JavaScript hoisting is the default behavior of moving the declaration on of the top.

In a JavaScript - We can use any variable and later we can declare. In the other words you can say that variable can be used before it has been declared.

Note: Function expression cannot be hoisted, only traditional function can be hoisted.

Note: Let & Var keyword also cannot be hoisted.
