What is Design Pattern ?

A design pattern are like a reusable solution for a common problems in software design. It's a high-level concept that can be customized to solve a recurring problem in code.

Design patterns are like blueprints that can be used to implement a solution that suits a program's needs.

Design patterns are different from algorithms, which define a clear set of actions to achieve a goal. Design patterns are more like high-level descriptions of a solution.
