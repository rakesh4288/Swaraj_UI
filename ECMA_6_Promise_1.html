What is Promises in JavaScript? Part I

Back to Dashboard

Promises is a new concept in JavaScript. Promise is an object that may produce two states condition:

  • Either Resolve or
  • Either Reject

While executing a promise sometimes it is possible to network failure Or may be huge traffic on internet so it will wait for the server response. Because Promise is very eager it will definitely be executed weather it will success or failed.

We can handle the resolve and reject status by using then and catch block. Yes, Promise providing the two method for handing the upcoming response.

We can use promise - when we have to perform some set of action and after that it will execute some or produce some result. And Promise is very eager it will definitely be executed whether the result will success or failed.After we can map the execution flow using .then() and .catch() callback function.
