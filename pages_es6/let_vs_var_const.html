Let vs Var vs Const

Back to Dashboard

Actually, both are using for creating a variable. Whereas Var keyword is the traditional approach and Let is the new keyword which is associated from ECMA6

The Main difference is that Let keyword has a block level scope and Var is the function level scope.

Note: Function expression cannot hoisted which is created by Let Keyword
