Spread Operator is a very simple and powerful feature introduced in the ES6 standard of the JavaScript, By using this feature we can write the nicer and shorter code.

The JavaScript spread operator is denoted by three dots (…)

The spread operator helps the iterable objects to expand into inside the individual elements. In other words, the spread operator allows us to copy all the elements from the existing array or object into another array or object.
