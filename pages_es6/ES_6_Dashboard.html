JavaScript ES6 New Features:

  • Class: New Keyword introduced
  • Extends: Extends keyword added while making inheritance from one of existing class
  • For Of: New Loop introduced and this will genetate the output like array indexing
  • Let Keyword: Let keyword came into picture and this is working same as Var Keyword. But here we have some difference like Var have a Function level of scope and Let have a block level of scope
  • Arrow Function: Arrow Function is also introduced in ECMA 6
  • Promise: Promise This is also new features of JavaScript. By using this we can get result either in Success Format or in a failed status.
  • Spread Operator: Spread Operator introduced in ES6 which is helping to convert array values into single string.
