map() creates a new array from calling a function for every array element.

map() does not execute the function for empty elements.

map() does not change the original array.

map() method of Array instances creates a new array populated with the results of calling a provided function on every element in the calling array.

why we use map function instead of for loop

map() and forEach() both are valuable methods for working with arrays in JavaScript.

Map Method:
  • Map method producing a new array without changing original array
  • And you can add any condition and that condition will iterate on every element
For Each Loop (forEach):

When we need to make any operation on existing array then we can use the for each method

within .map() we can also add new key value in existing array of object. Using ...spread operator we can perform this operation
