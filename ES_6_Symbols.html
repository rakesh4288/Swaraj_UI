What is a Symbols in ES6

Back to Dashboard
A JavaScript Symbol is a primitive datatype just like Number, String, Boolean, Object etc.
It represent the unique value that no other code can accidentally access.
So basically when you want to provide some unique value in the application then we can use the Symbols

This is the case sensitive. You need to use S capital letter while initializing the Symbol data type.
